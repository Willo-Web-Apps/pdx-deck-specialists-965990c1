# Design DNA — PDX Home Revival

## Visual Personality: warm-luxurious
## Layout: creative-portfolio
## Typography: elegant-serif

### Color Palette
- Primary (Forest Green): #2C4A3E → oklch(0.30 0.06 162)
- Accent (Cedar/Amber): #C17F3F → oklch(0.61 0.12 60)
- Surface (Linen): #F7F3EE → oklch(0.97 0.01 75)
- Text: oklch(0.20 0.03 162) — deep forest near-black

### Font Pairing
- Display: Cormorant Garamond — weight range 300–700 (luxury editorial serif)
- Body: Montserrat — weight range 300–600 (clean, refined sans-serif)

### Unique Design Choices
1. **Full-bleed cinematic hero** — linen-to-forest gradient with noise texture, oversized serif headline at 7xl–8xl creating an architectural firm aesthetic that competitors completely lack
2. **Named project spotlights** — "Lake Oswego Cedar Retreat" and "West Linn Pavilion" styled like architectural portfolio entries with caption-style typography, NOT generic contractor card grids
3. **Asymmetric trust section** — large amber accent numeral statistics (23+, 400+, $95K avg) set in Cormorant at massive scale against forest green background, evoking premium credibility without feeling like a cheap "stats bar"
4. **Warm linen section rhythm** — alternating linen/cream and subtle warm-gray panels create a tactile, magazine-quality page rather than the typical stark white contractor site
